Esse projeto é resultado de uma pesquisa para o trabalho de conclusão de curso do IFSP São Carlos. 

O trabalho foi motivado pela vontade de aplicar, na prática, os conhecimentos do curso por meio do desenvolvimento 
de um aplicativo funcional. A pandemia do COVID-19 destacou o papel dos dispositivos móveis em regiões com poucos recursos. 
Assim, o projeto propõe um aplicativo de apoio à análise de diagnósticos dermatológicos, ampliando o acesso à saúde em locais 
de difícil atendimento.

Para esse projeto contamos com o script disponível em:  https://github.com/syed-hamza/Skin-Cancer-Android-App, 
para o treinamento (Keras) e converter para tflite e utilizá-lo no aplicativo. 


Esse modelo classifica sete tipos de patologia:

1) Queratoses actínicas / Doença de Bowen
2) Carcinoma basocelular
3) Lesões benignas semelhantes à queratose
4) Dermatofibroma (nódulo benigno)
5) Melanoma (câncer de pele grave)
6) Nevo (pinta ou sinal benigno)
7) Nenhuma doença detectada
8) Lesões vasculares

O nome “Apy” que em Tupi Guarani é semelhante a "Leitura ou inspeção interna", é o mais próximo em linguagem nativa que encontrei 
para tentar comunicar "leitura de imagem interna" em um termo curto.
